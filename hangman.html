Hangman

August '17 – December’17
Tech Stack : Java

Hangman, an android app is a game made to simulate the classical game of Hangman.

It is a paper and pencil guessing game for two or more players. One player thinks of a word, phrase or sentence and the other(s) tries to guess it by suggesting letters within a certain number of guesses.

We developed the same idea into a game which can be played by multiple people using their mobile phones.

In addition, we have added various categories and background, which can be customized.
